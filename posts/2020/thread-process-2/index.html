More about process

Components

Process is an abstract concept contains how the program loaded to the memory and utilizing the cooresponding machine resources.

  • Process contains
    • Memories
      • text segment
      • data segment
      • heap segment
      • counter / processor / registers
      • stacks
    • OS resources

PCB

PCB is a kind of table stored each process and record the state and related information. While encounteriing external interruption, internal interruption, or software interruption, OS(CPU scheduler) will do the context switch between each process. And OS leverages PCB to manage each of the process.
